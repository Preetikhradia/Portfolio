# Portfolio
---

## 🚀 Run Locally

No setup needed. Just clone and open.

```bash
git clone https://github.com/preetikhradia/Portfolio.git
cd Portfolio
open index.html
```

Or drag `index.html` directly into your browser.

---

## 🔧 Customization Guide

All content lives inside `index.html`. Here's where to find what:

| What to change | Where to look |
|---|---|
| Name, role, tagline | `#hero` section |
| Work experience bullets | `#experience` section |
| Project titles, descriptions, GitHub links | `#projects` section — update `href="#"` with real URLs |
| Skills & certifications | `#skills` and `#certs` sections |
| Email & social links | `#contact` section + `nav-cta` in navbar |
| Accent color (gold) | `:root` → `--gold` CSS variable |
| Particle count / speed | `pCount` variable in the Three.js script block |

---

## 📸 Sections

- **Hero** — name, role, CTA buttons, live 3D background
- **Ticker** — scrolling stats bar
- **About** — summary + animated stat cards
- **Skills** — full tech stack in a filterable row layout
- **Experience** — timeline cards for SkyBrick & Infowiz
- **Projects** — CRM App, Finance Dashboard, ERP System
- **Certifications** — HackerRank, Cisco verified
- **Education** — DCRUST B.Tech with CGPA highlight
- **Contact** — direct email + social links
